# E-commerce Funnel Analysis

This is a simple student project for learning funnel analysis, data visualization, and machine learning with Python and Streamlit.

## Features
- Creates fake e-commerce data.
- Shows funnel conversion charts.
- Predicts whether a user will buy.
- Displays model accuracy and feature importance.
- Shows business insights in a dashboard.

## Files
- `01_funnel_analysis.py` - Main analysis script.
- `dashboard.py` - Streamlit dashboard.
- `data/fake_ecommerce_data.csv` - Saved fake dataset.
- `funnel_charts.png` - Funnel charts image.
- `confusion_matrix.png` - Model result image.

## Requirements
Install the packages in `requirements.txt`.

## Run the project
Run the analysis script:
```powershell
python 01_funnel_analysis.py
```

Run the dashboard:
```powershell
streamlit run dashboard.py
```

## Notes
- The data is fake and created for learning.
- The dashboard uses the saved CSV file from the analysis script.
